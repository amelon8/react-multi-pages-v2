# react-multi-pages-v2 (React + TypeScript + Vite)

This template provides a minimal setup to get React working in Vite with HMR and
some ESLint rules. It is set up with 4 pages: Home, About, Contact, Info

## Installing dependencies

```bash
cd react-multi-pages-v2
npm install
```

## Running app

```bash
cd react-multi-pages-v2
npm run dev
```
